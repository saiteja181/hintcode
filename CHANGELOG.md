# Changelog

All notable changes to HintCode will be documented in this file.

## [1.0.0] - 2026-03-20

### Added
- Progressive hint system (3 levels + solution reveal)
- Support for 5 coding platforms: LeetCode, HackerRank, CodeForces, CodeChef, GeeksforGeeks
- Auto-detection of problem statement, user code, and language
- Floating "Stuck? Get a Hint" button on problem pages
- Side panel UI for hints
- Manual mode for when auto-detection fails
- Hint caching per problem URL
- Options page with API key management and connection testing
- Popup with quick status overview
- Grok API integration (grok-3-mini)
- Privacy-first design (BYOK, no data collection)
